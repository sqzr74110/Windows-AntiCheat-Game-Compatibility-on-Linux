# Windows Anti-Cheat Game Compatibility on Linux

## 简介 | Introduction

**Windows Anti-Cheat Game Compatibility on Linux**  
是一个社区维护的记录项目，用于整理和记录：

> **在 Linux 系统环境下运行「带有反作弊系统的 Windows 游戏」的实际运行情况**

本项目不提供、不讨论任何绕过反作弊的方法，仅记录在不同 Linux 环境下的客观运行结果，供玩家与研究者参考。

---


## 📌 运行状态说明

- ✅ **可运行**：游戏可以正常启动并游玩
- ⚠️ **存在问题**：可以启动，但存在严重 BUG / 崩溃 / 性能问题
- ❌ **无法运行**：反作弊或其他原因阻止启动
- 🚫 **明确不支持**：反作弊机制从设计上不支持 Linux
- 🚨 **封号风险**：存在违反用户协议或封禁风险

---

## 🧪 测试环境示例格式

请尽量按照以下格式填写测试环境：

- 显示协议（Wayland / X11）
