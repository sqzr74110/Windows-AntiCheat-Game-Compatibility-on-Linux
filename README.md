# Windows Anti-Cheat Game Compatibility on Linux

本仓库用于记录 **在 Linux 系统环境下运行「带有反作弊系统的 Windows 游戏」的实际运行情况**。

⚠️ 本项目仅做兼容性与现象记录，不提供、不讨论任何绕过或破解反作弊的方法。

---

## 📊 Compatibility Table（运行情况表）

**本项目不使用 README 内的表格。**

👉 **所有数据请统一填写在 GitHub Projects 的表格中：**

🔗 **Compatibility Table (GitHub Project)**  
https://github.com/sqzr74110/Windows-AntiCheat-Game-Compatibility-on-Linux/projects

> 每一行代表一个游戏  
> `Title` 字段即 **Game Title（游戏名称）**

---

## ✍️ How to Contribute（如何填写）

请前往上方 **Projects 表格**，直接像填写 Excel 一样新增一行，并填写以下字段：

- **Title**：Game Title（游戏名称，英文）
- **Anti-Cheat System**：反作弊系统
- **Launch Method**：启动方式（Proton / Wine / Native 等）
- **Platform**：游戏平台
- **Status**：运行状态
- **Linux Environment**：测试环境
- **Notes**：备注说明

所有修改将自动保存，无需提交 Pull Request。

---

## 🏷️ Status Legend（状态说明）

- ✅ **Works**：可正常运行
- ⚠️ **Issues**：可运行但存在问题
- ❌ **Broken**：无法运行
- 🚨 **Ban Risk**：存在封号风险
